# Trikot-Ranking

Draggable Ranking-Tool für die 18 neuen Bundesliga-Heimtrikots der Saison 2026/27.

Trikots aus der Kammer links in die Tabelle rechts ziehen (oder antippen), um eine
eigene Rangfolge von Meister bis Absteiger zu bauen. Der Fortschritt wird lokal im
Browser gespeichert.

Einzelne, in sich geschlossene `index.html` (Bilder und Schriften als Data-URIs
eingebettet) – kein Build-Schritt nötig, direkt per GitHub Pages deploybar.

Trikotfotos: [footyheadlines.com](https://www.footyheadlines.com) (Presse-/Leak-Fotos).
Inoffizielles Fan-Projekt, nicht verbunden mit der DFL, der Bundesliga oder den
abgebildeten Vereinen.
